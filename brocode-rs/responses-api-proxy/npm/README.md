# @openai/brocode-responses-api-proxy

<p align="center"><code>npm i -g @openai/brocode-responses-api-proxy</code> to install <code>brocode-responses-api-proxy</code></p>

This package distributes the prebuilt [Brocode Responses API proxy binary](https://github.com/openai/brocode/tree/main/brocode-rs/responses-api-proxy) for macOS, Linux, and Windows.

To see available options, run:

```
node ./bin/brocode-responses-api-proxy.js --help
```

Refer to [`brocode-rs/responses-api-proxy/README.md`](https://github.com/openai/brocode/blob/main/brocode-rs/responses-api-proxy/README.md) for detailed documentation.
